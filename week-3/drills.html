Functions vs Function Calls

Resources: w3schools - Function Invocations
  1. function multiply1(x, y) { return x * y; } function multiply2(x, y) { x * y; }
    • What is the result of multiply1(5, 2) === 10? multiply2(5, 2) === 10?
    • What is the result of multiply1(5, 2) !== 10? multiply2(5, 2) !== 10?
    • What is the result of typeof multiply1(5, 2)? typeof multiply2(5, 2)?
    • What is the result of isNaN(multiply1(5, 2))? isNaN(multiply2(5, 2))?
    • What is the result of !isNaN(multiply1(5, 2))? !isNaN(multiply2(5, 2))?
    • What is the result of multiply1(5, 2) + multiply1(100,2)? multiply2(5, 2) + multiply2(100,2)?
    • What is the result of multiply1 + multiply2(5, 1)?
    • Create a statement using only the function multiply1 and + to compute 997
  2. var myObject = { firstName:"John", lastName: "Doe", fullName: function() { return this.firstName + " " + this.lastName; }, myself: function() { return this; } };
    • What is the result of myObject.firstName? myObject.lastName?
    • What is the result of myObject.fullName?
    • What is the result of myObject.fullName()?
    • What is the result of myObject.fullName('Jane', 'Smith')?
    • What is the result of myObject.myself?
    • What is the result of myObject.myself()?
    • What is the result of myObject.myself(23)?

Scopes & Variable Hoisting

If you get any of the following wrong after double-checking your answers in the console, review the following resource: Everything you wanted to know about javascript scope
  1. function foo() { var x = 3; console.log(x); } console.log(x);
    Run the above code. Explain what happened.
  2. var greeting = 'Hello'; var scope1 = function () { var scope2 = function () { var scope3 = function () { console.log(greeting); }; scope3(); }; scope2(); };
    What is the result of scope1()?
  3. var greeting = 'Hello'; var scope1 = function () { var scope2 = function () { var scope3 = function () { return greeting; }; return scope3(); }; return scope2(); };
    What is the result of scope1()?

    What is the result of scope1()?

Conditional Logic

  1. function bloop(x) { if (x === 'bloop') { return 'bleep'; } return 'blahp'; }
    • If bloop(someString) returns "bleep", what is the value of someString?
    • If bloop(someString2) returns "blahp", what do we know about someString2?
  2. function gloop(x) { if (x > 5) { x++; return x * -1; } else { x + 5; return x; } return x; }
    • Given a number num, if gloop(num) returns -10, what is num?
    • Given a number num2, if gloop(num2) returns -5, what is num2?
  3. Write a function isFoo that takes a string and returns "yep" if the string is "foo", and "nope" if the string is not "foo".
  4. Write a function divisibleByThree that takes an integer and returns the string "is divisible by three" if it is divisible by 3; or it returns "is NOT divisible by three" if it is not divisible by 3.
  5. Write a function divisibleByFive that is like divisibleByThree but for the number 5 instead of 3.
  6. Write a function
    function divisibleBy(divisor, int) { }
    that determines if integer int is divisible by divisor. Make it work for any integer divisor and any integer int.
  7. Write a function isANumber1 that takes some argument, and returns true if it is a number, or false if it is not a number. Use the method isNaN.
  8. Write a function isANumber2 that takes some argument, and returns true if it is a number, or false if it is not a number. Do NOT use the method isNaN.
  9. Write a function isInteger that takes a number, and returns true if it is an integer, or false if it is not an integer.
  10. Write a function newDivisibleBy that does the same thing as divisibleBy when divisor and int are integers, but that throws an error message "Arguments must be integers" if either int OR divisor are not an integer. An integer is 1) a number that 2) has a modulo 0 (% 0) of zero (i.e., x % 1 === 0).
